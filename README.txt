**README - Flappy Penguin CSS Animation Project (Guided by freeCodeCamp)**

### English:

#### Introduction:
Welcome to the Flappy Penguin CSS Animation Project, proudly guided by freeCodeCamp! Dive deep into the mesmerizing world of CSS animations with this interactive project and bring the adorable flappy penguin to life. Designed for beginners and those eager to hone their CSS skills, this project provides an excellent gateway into the realm of CSS animations and transformations.

#### About freeCodeCamp:
freeCodeCamp is an esteemed open-source community that offers free coding lessons and projects to help learners acquire coding skills. This particular project is one of the many resources provided by them to guide learners in understanding CSS animations.

#### How it Works:
1. The scene sets a penguin amidst a backdrop of serene mountains and a radiant sun.
2. Engage with the penguin: Clicking on it triggers the `scale` transformation, causing it to enlarge.
3. Notice the penguin's left arm? It showcases a waving animation, breathing life into the scene.
4. The environment captures a tranquil setting of the sun casting its glow behind mountains.

#### Project Structure:
- **HTML**: Houses the penguin and its ambient backdrop.
- **CSS**: Contains the styles, animations, and transformations vital for the scene's vibrancy.

#### How to Use:
1. Clone or download the repository.
2. Open the HTML file in your browser of choice.
3. Enjoy the captivating animations! Feel free to delve into the CSS file to grasp the intricacies and adjust as per your curiosity.

### Français:

#### Introduction:
Bienvenue dans le projet d'animation CSS Flappy Penguin, fièrement guidé par freeCodeCamp! Plongez au cœur du monde envoûtant des animations CSS avec ce projet interactif et donnez vie au charmant pingouin. Conçu pour les débutants et ceux qui souhaitent perfectionner leurs compétences en CSS, ce projet offre une excellente introduction au monde des animations et transformations CSS.

#### À propos de freeCodeCamp:
freeCodeCamp est une communauté open-source estimée qui propose des leçons et projets de codage gratuits pour aider les apprenants à acquérir des compétences en codage. Ce projet en particulier est l'une des nombreuses ressources qu'ils offrent pour guider les apprenants dans la compréhension des animations CSS.

#### Comment ça marche:
1. La scène présente un pingouin sur fond de montagnes paisibles et d'un soleil radieux.
2. Interagissez avec le pingouin: en cliquant dessus, vous déclencherez la transformation `scale`, le faisant agrandir.
3. Avez-vous remarqué le bras gauche du pingouin? Il présente une animation de salut, ajoutant du dynamisme à la scène.
4. L'environnement représente un cadre tranquille avec le soleil projetant sa lueur derrière les montagnes.

#### Structure du projet:
- **HTML**: Contient le pingouin et son environnement pittoresque.
- **CSS**: Intègre les styles, animations et transformations qui donnent vie à la scène.

#### Comment utiliser:
1. Clonez ou téléchargez le dépôt.
2. Ouvrez le fichier HTML dans votre navigateur préféré.
3. Admirez les animations envoûtantes! N'hésitez pas à explorer le fichier CSS pour en comprendre les subtilités et le modifier selon votre curiosité.
